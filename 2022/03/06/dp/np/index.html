Data Type

為了能進行高速運算，並確保計算時值的精度，有資料的「型別」(type)

int (signed)

  • int8 : 具有正負號的8位元整數
  • int16 : 具有正負號的16位元整數
  • int32 : 具有正負號的32位元整數
  • int64 : 具有正負號的64位元整數

uint (unsigned)

  • uint8 : 沒有正負號的8位元整數
  • uint16 : 沒有正負號的16位元整數
  • uint32 : 沒有正負號的32位元整數
  • uint64 : 沒有正負號的64位元整數

float

  • float16 : 16位元的浮點數
  • float32 : 32位元的浮點數
  • float64 : 64位元的浮點數
