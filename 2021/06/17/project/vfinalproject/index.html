遊戲機制

蛇吃完三個蘋果即為獲勝，若是遊戲途中不慎觸碰到邊界即為失敗。

功能鍵

  1. switch0 - reset
  2. button2 - left
  3. button1 - start
  4. button0 - right

遊戲機制設定

因為遊戲只能靠左右鍵控制蛇的方向，再加上蛇的移動速度較快，故不容易獲勝。
